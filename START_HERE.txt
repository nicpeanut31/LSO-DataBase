START HERE — ADMINISTRATOR FULL ACCESS / STAFF READ ONLY
=======================================================

1. Download a Complete Backup from the current live website.
2. Open Supabase > SQL Editor > New query.
3. Run ADMIN_READ_ONLY_SQL_UPDATE.sql from this folder.
4. Replace the files in the GitHub Pages repository with every website file in
   this folder. Do not rename the files.
5. Commit and wait for GitHub Pages deployment.
6. Open the website and press Ctrl+F5.
7. Log in as Administrator and confirm adding/editing works.
8. Log in as a Staff Account and confirm that viewing/printing works while all
   write controls are hidden or disabled.

The configured Supabase URL and publishable key are retained.
