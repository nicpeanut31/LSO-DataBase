LSO GITHUB UPLOAD INSTRUCTIONS
==============================

1. Extract this ZIP.
2. Open the extracted folder.
3. Upload the FILES INSIDE the folder to the GitHub repository publishing root.
4. Do not upload the ZIP itself.
5. Keep every filename exactly as provided. Do not add (1), (2), or similar suffixes.
6. Confirm that index.html is in the repository root used by GitHub Pages.
7. Commit the upload and wait for GitHub Pages to finish deploying.
8. Open the website and press Ctrl + Shift + R once.
9. If the PWA is installed, close it completely and reopen it after the hard refresh.

This package deliberately excludes old test screenshots, prior-version scripts,
old service workers, duplicate update notes, and previous build reports.
