OFFICIAL PDF PRINT TEMPLATE UPDATE
==================================

The uploaded LSO official PDF is now the default print container for every
report generated by the website.

APPLIES TO
----------
- Official Member Record
- Individual Attendance Report
- Semester Attendance Report
- Monthly Attendance Report
- Overall Attendance Report
- Individual Duty Hours Report
- Current Trainee Duty Hours Report
- Current Probationary Duty Hours Report
- Monthly Trainee Duty Hours Report
- Monthly Probationary Duty Hours Report
- Overall Duty Hours Report

PRINT BEHAVIOR
--------------
- Every printed page repeats the exact official PDF header and footer.
- No additional print logo is inserted because the official template already
  contains the LSO identity.
- All reports are generated in A4 portrait orientation.
- Wide tables are compacted, wrapped, and automatically continued on duplicate
  template pages when more space is needed.
- Table headings repeat on subsequent pages.
- Redundant system-generated footer messages are removed because the official
  template already provides the document footer.

FILES ADDED
-----------
- lso-official-template.pdf      Original uploaded official template
- lso-official-template.png      Full-page rendered reference
- lso-official-header.png        Exact official header used on every page
- lso-official-footer.png        Exact official footer used on every page

INSTALLATION
------------
Replace all GitHub website files with this package. Keep the new PDF and PNG
assets beside index.html. No Supabase SQL update is required.
