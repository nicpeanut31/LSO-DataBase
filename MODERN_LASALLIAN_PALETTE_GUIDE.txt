DEPLOYMENT
----------
1. Extract the ZIP.
2. Upload every file and folder inside this package to the GitHub Pages publishing root.
3. Replace the existing index.html.
4. Wait for GitHub Pages to finish deploying.
5. On desktop, use Ctrl + Shift + R.
6. On mobile or the installed PWA, close the app completely, reopen it while online, and refresh once.
7. When an older palette remains, open refresh-lso.html once to unregister the previous service worker and clear old LSO caches.

DATABASE SAFETY
---------------
No Supabase table, SQL schema, database record, role permission, Attendance workflow, Duty Hours workflow, account authentication, or contract/report logic was changed by this design update.
